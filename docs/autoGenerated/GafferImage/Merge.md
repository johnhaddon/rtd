# Merge

Composites two or more images together. The following operations
are available :

  - Add : A + B
  - Atop : Ab + B(1-a)
  - Divide : A / B
  - In : Ab
  - Out : A(1-b)
  - Mask : Ba
  - Matte : Aa + B(1.-a)
  - Multiply : AB
  - Over : A + B(1-a)
  - Subtract : A - B
  - Difference : fabs( A - B )
  - Under : A(1-b) + B
